本文档，对此此工程增加了一些说明，方便快速上手。
接线：
PD6<------------------------>PD6
PD7<------------------------>PD7
PA4<------------------------>PA4
其中：PD6，PD7为I2C口，PA4为消息通知口。
可通过iic_demo.h的IIC_MASTER_MODE来配置主从，分别烧录到两个fr801xH系列的模块中。
流程：
1、编译主从程序。
2、烧录到对应模块。
3、接线。
4、先开机从机，再开主机。此时程序就会主从开始通信。此程序为乒乓程序，主往从机发送数据后，从机接收到数据往主机发送数据。
